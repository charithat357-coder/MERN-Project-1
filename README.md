# MERN Marks Portal

A role-based marks management portal built with the MERN stack.  
Provides secure login, dashboards for students/faculty/HOD/admin, and real-time marks updates.

## Features
-  Role-based authentication
-  Dashboards for each user type
-  Notification system
- Responsive UI with light theme

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
